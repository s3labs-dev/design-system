The s3 labs icon system uses consistent 24px icons with a 2px stroke weight. Icons are rendered in accent colors to match the bio-luminescent aesthetic.

## CSS Implementation

```css
.icon {
  width: 24px;
  height: 24px;
  stroke: currentColor;
  stroke-width: 2;
  stroke-linecap: round;
  stroke-linejoin: round;
  fill: none;
}

/* Color variants */
.icon-green { color: var(--s3-bio-green); }
.icon-cyan { color: var(--s3-glow-cyan); }
.icon-gold { color: var(--s3-bio-gold); }
.icon-muted { color: var(--s3-moss); }

/* Size variants */
.icon-sm { width: 16px; height: 16px; }
.icon-lg { width: 32px; height: 32px; }
```

## Usage Guidelines

- Use icons to supplement text, not replace it
- Maintain consistent sizing within a context
- Apply accent colors purposefully (green for actions, cyan for info, gold for warnings)
- Icons in buttons should be 16-20px
- Standalone icons should be 24px

## Icon Sources

The s3 labs icons are based on [Lucide](https://lucide.dev/), an open-source icon library. Custom icons follow the same 24px/2px specifications.
