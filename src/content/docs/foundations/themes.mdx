The s3 labs design system supports both dark (default) and light themes. The dark theme embodies the "deep forest" aesthetic, while light mode adapts colors for high-brightness contexts.

## Dark Mode (Default)

The default theme features deep forest backgrounds with bio-luminescent accents.

## Theme CSS Variables

### Dark Mode

```css
:root {
  /* Accent */
  --sl-color-accent-low: #1a2a15;
  --sl-color-accent: #7ed957;
  --sl-color-accent-high: #b8f09a;

  /* Gray Scale */
  --sl-color-gray-1: #f5f1e8;
  --sl-color-gray-2: #e8e4db;
  --sl-color-gray-3: #6a7858;
  --sl-color-gray-4: #4a5942;
  --sl-color-gray-5: #3d4a35;
  --sl-color-gray-6: #242b1f;
  --sl-color-gray-7: #0d100b;

  /* Backgrounds */
  --sl-color-bg: #0d100b;
  --sl-color-bg-nav: rgba(13, 16, 11, 0.85);
  --sl-color-bg-sidebar: #141912;
}
```

### Light Mode

```css
:root[data-theme='light'] {
  /* Accent - Darkened for contrast */
  --sl-color-accent-low: #e8f5e0;
  --sl-color-accent: #4a6830;
  --sl-color-accent-high: #2d4020;

  /* Gray Scale - Inverted */
  --sl-color-gray-1: #0d100b;
  --sl-color-gray-2: #1a1f16;
  --sl-color-gray-3: #4a5942;
  --sl-color-gray-4: #6a7858;
  --sl-color-gray-5: #a1a89a;
  --sl-color-gray-6: #d4d7cc;
  --sl-color-gray-7: #f5f1e8;

  /* Backgrounds */
  --sl-color-bg: #f5f1e8;
  --sl-color-bg-nav: rgba(245, 241, 232, 0.9);
  --sl-color-bg-sidebar: #e8e4db;
}
```

## Ambient Background

The ambient orb effect adapts between themes:

```css
/* Dark Mode */
body::before {
  background:
    radial-gradient(ellipse 600px at 10% 5%, rgba(126, 217, 87, 0.06), transparent),
    radial-gradient(ellipse 500px at 90% 40%, rgba(95, 251, 241, 0.04), transparent),
    radial-gradient(ellipse 450px at 30% 90%, rgba(212, 165, 116, 0.04), transparent);
}

/* Light Mode */
:root[data-theme='light'] body::before {
  background:
    radial-gradient(ellipse 600px at 10% 5%, rgba(74, 104, 48, 0.04), transparent),
    radial-gradient(ellipse 500px at 90% 40%, rgba(42, 138, 130, 0.03), transparent),
    radial-gradient(ellipse 450px at 30% 90%, rgba(138, 101, 56, 0.03), transparent);
}
```

## Theme Toggle

Starlight handles theme switching automatically with a toggle in the header. The theme preference is persisted to localStorage.

## Implementation Notes

- Always test components in both themes
- Use CSS custom properties for all colors
- Glow effects should be reduced or removed in light mode
- Ensure minimum 4.5:1 contrast ratio for text in both themes
