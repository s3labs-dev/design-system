Buttons are the primary interactive elements in the s3 labs design system. They feature bio-luminescent glow effects on hover.

### Primary

The primary button is used for main actions. It uses the Bio-Green accent color with a glow effect on hover.

```css
.btn-primary {
  background: var(--s3-bio-green);
  color: var(--s3-void);
}

.btn-primary:hover {
  background: var(--s3-bio-green-bright);
  box-shadow: var(--s3-glow-green);
}
```

### Secondary

Secondary buttons have a transparent background with a Bio-Green border. Use for secondary actions.

```css
.btn-secondary {
  background: transparent;
  color: var(--s3-bio-green);
  border: 1px solid var(--s3-bio-green);
}

.btn-secondary:hover {
  background: rgba(126, 217, 87, 0.1);
}
```

### Ghost

Ghost buttons have a subtle background tint. Use for tertiary actions or in dense UI areas.

```css
.btn-ghost {
  background: rgba(126, 217, 87, 0.1);
  color: var(--s3-bio-green);
}

.btn-ghost:hover {
  background: rgba(126, 217, 87, 0.2);
}
```

## Accessibility

- Buttons maintain a minimum contrast ratio of 4.5:1 for text
- Focus states use a 2px Bio-Green outline with 2px offset
- Disabled buttons reduce opacity to 0.5 and remove cursor interaction
