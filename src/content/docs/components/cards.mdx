## Card Anatomy

| Element | Style |
|---------|-------|
| Background | `--s3-void` or `--s3-forest-base` |
| Border | 1px rgba(126, 217, 87, 0.1) |
| Border Radius | `--s3-radius-lg` (20px) |
| Padding | `--s3-space-5` (24px) |
| Hover Border | rgba(126, 217, 87, 0.25) |
| Hover Glow | `--s3-glow-green` |
| Hover Transform | translateY(-2px) |

## CSS Implementation

```css
.card {
  background: var(--s3-void);
  border: 1px solid rgba(126, 217, 87, 0.1);
  border-radius: var(--s3-radius-lg);
  padding: var(--s3-space-5);
  transition: all var(--s3-duration-normal) var(--s3-ease-out);
}

.card:hover {
  border-color: rgba(126, 217, 87, 0.25);
```

## Card Variants

### Elevated Card

Uses `--s3-forest-base` background for higher elevation.

```css
.card-elevated {
  background: var(--s3-forest-base);
}
```

### Interactive Card

For clickable cards, add cursor and scale effect.

```css
.card-interactive {
  cursor: pointer;
}

.card-interactive:hover {
  transform: translateY(-4px) scale(1.01);
}

.card-interactive:active {
  transform: translateY(-2px) scale(0.99);
}
```

### Accent Border Card

Adds a colored top border for categorization.

```css
.card-accent-green {
  border-top: 3px solid var(--s3-bio-green);
}

.card-accent-cyan {
  border-top: 3px solid var(--s3-glow-cyan);
}

.card-accent-gold {
  border-top: 3px solid var(--s3-bio-gold);
}
```

## Card Grid

```css
.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
  gap: var(--s3-space-5);
}
```

## Usage Guidelines

- Use cards to group related content
- Keep card content scannable (title, brief description, action)
- Maintain consistent padding and spacing within card grids
- Use hover effects to indicate interactivity
