## Token Categories

### Colors by Category

#### Foundation (Backgrounds & Surfaces)

| Token | Value | Usage |
|-------|-------|-------|
| `--s3-void` | #0d100b | Primary background |
| `--s3-void-elevated` | #141912 | Sidebar, nav |
| `--s3-forest-base` | #242b1f | Cards, panels |
| `--s3-forest-mid` | #2d3628 | Hover states |
| `--s3-moss-dark` | #3d4a35 | Borders |

#### Accents (Interactive & Emphasis)

| Token | Value | Usage |
|-------|-------|-------|
| `--s3-bio-green` | #7ed957 | Primary CTA |
| `--s3-glow-cyan` | #5ffbf1 | Links, info |
| `--s3-bio-gold` | #d4a574 | Warnings |

#### Text

| Token | Value | Usage |
|-------|-------|-------|
| `--s3-cream` | #f5f1e8 | Primary text |
| `--s3-cream-dim` | #e8e4db | Body text |
| `--s3-moss-bright` | #6a7858 | Secondary text |
| `--s3-moss` | #4a5942 | Muted text |

## SCSS Variables

For Sass/SCSS projects:

```scss
// Colors
$s3-void: #0d100b;
$s3-forest-base: #242b1f;
$s3-bio-green: #7ed957;
$s3-glow-cyan: #5ffbf1;
$s3-bio-gold: #d4a574;
$s3-cream: #f5f1e8;

// Spacing
$s3-space: (
  1: 4px,
  2: 8px,
  3: 12px,
  4: 16px,
  5: 24px,
  6: 32px,
  7: 48px,
  8: 64px
);

// Function to get spacing
@function space($key) {
  @return map-get($s3-space, $key);
}
```
