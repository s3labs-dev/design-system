## Fonts

The s3 labs design system uses three font families, all available from Google Fonts.

### Google Fonts Import

Add this to your CSS:

```css
@import url('https://fonts.googleapis.com/css2?family=Outfit:wght@300;400;500;600;700;800;900&family=JetBrains+Mono:wght@400;500;600&family=Fraunces:ital,opsz,wght@0,9..144,300;0,9..144,400;0,9..144,500;0,9..144,600;1,9..144,400&display=swap');
```

### HTML Link

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Outfit:wght@300;400;500;600;700;800;900&family=JetBrains+Mono:wght@400;500;600&family=Fraunces:ital,opsz,wght@0,9..144,300;0,9..144,400;0,9..144,500;0,9..144,600;1,9..144,400&display=swap" rel="stylesheet">
```

## Brand Guidelines

### Logo Clear Space

Maintain clear space around the logo equal to the height of one bar (~25% of total height).

### Minimum Sizes

- **Digital**: 32px height minimum
- **Print**: 12mm height minimum

### Approved Backgrounds

- Void (#0d100b)
- Cream (#f5f1e8)
- Forest gradients
- Any solid dark color with sufficient contrast

### Prohibited Uses

- Don't rotate or skew the logo
- Don't change gradient colors
- Don't add effects (shadows, glows, outlines)
- Don't place on busy backgrounds
- Don't stretch or compress
