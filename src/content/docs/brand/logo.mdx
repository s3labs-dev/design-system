## Usage Guidelines

### Clear Space

Maintain a minimum clear space around the logo equal to the height of one bar (approximately 1/4 of the total mark height). This ensures the logo maintains its visual impact.

### Minimum Size

- **Digital**: 32px minimum height
- **Print**: 12mm minimum height

### Do's

- Use the tri-color version as the primary mark
- Use mono-green for constrained spaces (avatars, favicons)
- Use the light variant on cream/white backgrounds

### Don'ts

- Don't rotate the logo
- Don't change the gradient colors
- Don't add effects (drop shadows, glows) to the mark itself
- Don't stretch or distort proportions

## Horizontal Lockup

For headers and navigation, use the horizontal lockup with the mark and wordmark:

```
[Mark] s3·labs
```

The wordmark uses:
- **s3**: Outfit 800, Bio-Green (#7ed957)
- **·**: Moss dot separator (#4a5942)
- **labs**: Outfit 300, Cream (#f5f1e8)
