## Semantic Colors

| Purpose | Variable | Value |
|---------|----------|-------|
| Success | `--s3-success` | `var(--s3-bio-green)` |
| Warning | `--s3-warning` | `var(--s3-bio-gold)` |
| Info | `--s3-info` | `var(--s3-glow-cyan)` |
| Error | `--s3-error` | `#ef6b6b` |

## Glow Effects

Each accent color has a corresponding glow shadow for hover states:

```css
--s3-glow-green: 0 0 40px rgba(126, 217, 87, 0.15);
--s3-glow-cyan: 0 0 40px rgba(95, 251, 241, 0.15);
--s3-glow-gold: 0 0 40px rgba(212, 165, 116, 0.15);
```

## Light Mode Adaptations

In light mode, the accent colors are darkened for contrast on cream backgrounds:

| Dark Mode | Light Mode |
|-----------|------------|
| `#7ed957` | `#4a6830` |
| `#5ffbf1` | `#2a8a82` |
| `#d4a574` | `#8a6538` |
