The s3 labs type system uses three complementary font families: **Outfit** for display and body text, **JetBrains Mono** for code and technical labels, and **Fraunces** for editorial accents.

## Font Families

### Outfit — Display & Body

A geometric sans-serif with a technical feel and excellent readability. Used for all headings and body text.

```css
--s3-font-display: 'Outfit', sans-serif;
--s3-font-body: 'Outfit', sans-serif;
```

### JetBrains Mono — Code & Labels

A monospace font optimized for code with increased height for better readability. Used for code blocks, technical labels, and badges.

```css
--s3-font-mono: 'JetBrains Mono', monospace;
```

### Fraunces — Editorial Accent

A variable serif with an organic, hand-drawn quality. Used sparingly for taglines, quotes, and editorial touches.

```css
--s3-font-accent: 'Fraunces', serif;
```

## Weight Scale

Outfit provides a full range of weights:

| Weight | Value | Usage |
|--------|-------|-------|
| Light | 300 | Wordmark "labs" |
| Regular | 400 | Body text |
| Medium | 500 | Emphasis |
| Semibold | 600 | Headings H1-H2 |
| Bold | 700 | Display 2, strong emphasis |
| Extrabold | 800 | Display 1, wordmark "s3" |

## Letter Spacing

Negative letter spacing is used for display sizes to create tighter, more impactful headlines:

| Size | Letter Spacing |
|------|----------------|
| Display 1 | -0.04em |
| Display 2 | -0.03em |
| Heading 1 | -0.02em |
| Heading 2 | -0.01em |
| Body | normal |
| Mono Labels | 0.05em - 0.15em |

## CSS Implementation

```css
/* Display 1 */
.display-1 {
  font-family: var(--s3-font-display);
  font-size: 72px;
  font-weight: 800;
  letter-spacing: -0.04em;
  line-height: 1.05;
}

/* Display 2 */
.display-2 {
  font-family: var(--s3-font-display);
  font-size: 48px;
  font-weight: 700;
  letter-spacing: -0.03em;
  line-height: 1.1;
}

/* Heading 1 */
.heading-1 {
  font-family: var(--s3-font-display);
  font-size: 32px;
  font-weight: 600;
  letter-spacing: -0.02em;
  line-height: 1.2;
}

/* Body */
.body {
  font-family: var(--s3-font-body);
  font-size: 16px;
  font-weight: 400;
  line-height: 1.7;
}

/* Mono Label */
.mono-label {
  font-family: var(--s3-font-mono);
  font-size: 11px;
  font-weight: 600;
  letter-spacing: 0.1em;
  text-transform: uppercase;
}

/* Accent */
.accent {
  font-family: var(--s3-font-accent);
  font-size: 20px;
  font-weight: 400;
  font-style: italic;
  line-height: 1.5;
}
```
